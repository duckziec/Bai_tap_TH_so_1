## 📖 Mô tả

PVHub là một trang web học trực tuyến được thiết kế theo phong cách các nền tảng LMS như Udemy hoặc Azota. Giao diện sử dụng tông màu trắng – xanh theo logo, bố cục đơn giản, hiện đại và dễ đọc.

Project tập trung vào việc xây dựng trang chủ của một hệ thống học tập trực tuyến.

---

## 🎯 Mục tiêu bài tập

* Thực hành bố cục layout

## 🚀 Chạy project local

Clone project:

```bash
git clone https://github.com/your-username/your-repository.git
```

Di chuyển vào folder:

```bash
cd your-repository
```

Cài dependencies:

```bash
npm install
```

Chạy project:

```bash
npm run dev
```

Build production:

```bash
npm run build
```
